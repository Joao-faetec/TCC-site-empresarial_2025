Sobre o Projeto

O Sistema Bibliotecário Faetec é uma solução completa desenvolvida para gerenciar todas as operações de uma biblioteca escolar. O sistema foi especificamente projetado para atender às necessidades da Faetec, uma instituição de ensino de excelência.

Com uma interface intuitiva e funcionalidades robustas, o sistema permite o controle eficiente do acervo, registro de empréstimos e devoluções, gerenciamento de multas, e geração de relatórios detalhados. O objetivo é proporcionar uma experiência fluida tanto para bibliotecários quanto para usuários finais.

A plataforma foi desenvolvida seguindo as melhores práticas de engenharia de software, garantindo segurança, performance e escalabilidade para atender às demandas de uma instituição educacional.

Funcionalidades Principais

  • Cadastro de Livros: Sistema completo para adicionar, editar e remover livros do acervo com informações detalhadas como ISBN, autor, editora e data de publicação.
  • Controle de Empréstimos: Registro automático de empréstimos com datas de devolução, renovações e histórico de transações.
  • Gerenciamento de Devoluções: Processamento de devoluções com cálculo automático de multas por atraso.
  • Reservas de Livros: Sistema de reserva que permite aos usuários reservar livros indisponíveis.
  • Categorização e Busca: Organização por categorias, gêneros e autores com busca avançada e filtros.
  • Relatórios Detalhados: Geração de relatórios sobre acervo, empréstimos, usuários mais ativos e livros mais populares.

Detalhes Técnicos

Backend: O sistema foi desenvolvido em PHP seguindo o padrão MVC (Model-View-Controller), garantindo uma arquitetura limpa e manutenível. Utiliza MySQL como banco de dados relacional para armazenar informações de livros, usuários e transações.

Frontend: A interface foi construída com HTML5, CSS3 e JavaScript puro, com Bootstrap 4 para garantir responsividade e design moderno. A experiência do usuário foi otimizada para funcionar em diferentes dispositivos.

Segurança: O sistema implementa práticas de segurança robustas incluindo validação de entrada, proteção contra SQL injection, autenticação segura e controle de acesso baseado em papéis.
